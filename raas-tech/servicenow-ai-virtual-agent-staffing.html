Roles We Place

Virtual Agent Developer
Contract / Full-Time

Designs and builds ServiceNow Virtual Agent conversation flows across IT, HR, and customer service use cases. Develops decision trees, fulfillment integrations, and handoff logic that deflect tickets from the service desk while delivering a seamless user experience. Continuously optimizes conversation paths based on analytics and user feedback.

NLU Specialist
Contract / Full-Time

Trains and optimizes Natural Language Understanding models within ServiceNow to improve intent recognition accuracy across Virtual Agent topics. Builds and maintains NLU model training datasets, evaluates entity extraction performance, and tunes confidence thresholds to balance deflection rates with user experience quality metrics.

Now Assist Implementation Consultant
Contract / Full-Time

Implements ServiceNow Now Assist generative AI features across ITSM, HR, and CSM workflows — including AI-powered case summarization, resolution notes, and agent coaching. Configures prompt engineering guidelines, integrates with enterprise LLM providers, and builds governance frameworks to ensure AI outputs meet enterprise quality and compliance standards.

AI Search & Predictive Intelligence Developer
Contract / Full-Time

Configures ServiceNow AI Search and Predictive Intelligence to surface relevant knowledge, automate classifications, and recommend solutions to agents and end users. Builds machine learning solution definitions, trains classification models on historical incident and request data, and measures model accuracy to drive continuous improvement.
